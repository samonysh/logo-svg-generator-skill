# NovaSync — Cross-device file sync tool

Design a logo for **NovaSync**, a cross-device file sync tool for developers.
It syncs folders between desktop, mobile, and cloud with zero config.

- **Audience:** developers, power users
- **Vibe:** clean, geometric, trustworthy, modern, technical
- **Avoid:** playful, childish, hand-drawn, gradients
- **Preferred color:** indigo `#4F46E5`
- **Deliverables:** monochrome master + branded variant, sizes 16 / 24 / 32 / 64 / 128 / 256 / 512 px
- **Metaphor hints:** letter "N" combined with sync arrows loop, or two orbiting nodes

Generate the SVG set following Simple Icons rules (single path, no fill on master, touches ≥ 2 sides of the 24×24 viewBox, centered).
