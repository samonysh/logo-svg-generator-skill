# LumenDB — Realtime analytics database

Design a logo for **LumenDB**, a realtime analytics database that ingests event streams and returns sub-second query answers.

- **Audience:** data engineers, backend teams
- **Vibe:** precise, luminous, fast, engineered
- **Avoid:** biotech, medical, food
- **Preferred color:** teal `#0EA5A5`
- **Deliverables:** monochrome master + branded variant, standard multi-size set
- **Metaphor hints:** prism refracting into a data spark, or a bar-chart "L" made of light

Follow Simple Icons rules (single `<path>`, no fill on master, touches ≥ 2 sides of the 24×24 viewBox, visually centered, ink coverage 25–65 %).
